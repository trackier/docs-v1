# List Advertisers

| **HTTP method** | End Point |
| :--- | :--- |
| **GET** | **/advertiser** |
| GET | /advertiser |
| GET | /advertiser/{id} |

## **Response body parameters**

* _**Get all the advertisers**_

```bash
curl -X GET -H "X-Api-Key: {key}" -H "Cache-Control: no-cache" "http://api.trackier.com/advertiser"
```
