# Delete a custom field

## Affiliates

| **HTTP Method** | **End Point** |
| :--- | :--- |
| DELETE | /dictionary/customField/affiliate/{id} |

### Example Request

```bash
curl -X DELETE -H "X-Api-Key: {key}" -H "Cache-Control: no-cache" "http://api.trackier.com/dictionary/customField/affiliate/{id}"
```

### Response Body

```javascript
{
    "data": {
        "message": "string"
    },
    "success": false
}
```
