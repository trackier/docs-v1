# Campaigns

| **HTTP method** | **End Point** | **Description** |
| :--- | :--- | :--- |
| GET | /advertiser/campaigns | Get all the campaigns |
| GET | /advertiser/campaigns/{campaignId} | Get info about a specific campaign |

## **All Campaigns**

### **Sample Request**

```bash
curl -X GET -H "X-Api-Key: {key}" -H "Cache-Control: no-cache" -H "Postman-Token: f589a813-1ead-93b8-1af7-c57c74376a66" "https://api.trackier.com/advertiser/campaigns"
```
